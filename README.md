# ExpressDeluxe
Technologies Used:
* Node/Express
* CouchDB
* React/Alt
* Bootstrap

Need Node/npm installed

Install these packages globally

`npm install gulp -g`
  
`npm install bower -g`
  
`npm install`
  
`bower install`

NPM Packages:
 * alt
 * react
 * bower
 * browserify
 * uglify
 * winston

Development Packages:
